# run for cmd
as the name suggest, this is the command line implementation of windows run dialog. you can download it [here](https://github.com/salman-bermano/run-for-cmd/releases/latest)

# how to use
if the program you're trying to run is placed in the system32 folder or windows environment, then you can simply run the program like this:
```batchfile
run notepad
```
you can also use the full path to the program/file you're trying to run:
```batchfile
run c:\Windows\System32\notepad.exe
```
